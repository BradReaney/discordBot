# Discord bot for Obi WAN 2: Ubuntu Boogaloo server

## Setup

Prerequasits

1. Node JS 16 >
2. .env file with the follwoing variables
    - DISCORD_TOKEN=""
    - CLIENT_ID=""
    - GUILD_ID=""

Install dependencies - `npm install`
Run bot - `npm start`
